# EDDLocalOrganicView
Plugin for EDDiscovery. Shows organic signals of planets in the current star system.

# Description
На создание плагина для популярной программы EDDiscovery автора сподвигло отсутствие нормальных альтернатив для отображения органических сигналов (экзобиология) в этой программе. Можно запросить историю сканирования, но это отнимает большое количество времени. Программа надолго зависает, особенно, если вы плотно занимаетесь экзобиологией и у вас несколько сотен, а то и тысяч записей в журнале сканирования. Автору захотелось иметь только локальную информацию по конкретной системе, чтобы не перескакивать из режима в режим в самой игре и не ххранить в памяти информацию о типах органики, что отсканировано, а что нет и так далее.

# Setup
Установка плагина в EDDiscovery предельно проста. Необходимо скопировать #DVGSoft.LocalOrganicScanViewer.dll# в папку %%LOCALAPPDATA%% \ EDDiscovery \DLL, если у вас полноценная установленная версия (не проверено, автор пользуется Portable версией), или %%PATH_TO_EDDISCOVERY%%\Data\DLL, если вы используете Portable версию программы.
Подробнее: https://github.com/EDDiscovery/EDDiscovery/wiki/5.1-Extending-EDD-with-DLLs
Плагин требует установки .NET Framework 4.8 для своей работы.
